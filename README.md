# Shadow Tail

一人称視点風の尾行ゲームです。

プレイヤーは前方を歩く人物を追跡します。近づきすぎると怪しまれ、離れすぎると見失います。対象が振り返ったときは、電信柱・ポスト・看板・自販機などの遮蔽物に隠れて発見を避けます。

尾行対象はプレイヤーの操作に関係なく移動します。止まっていると対象との距離が開いていくため、適度に前進して追い続ける必要があります。

## ゲームの目的

対象が落としたアイテムをすべて回収するとミッションクリアです。

失敗条件は主に次の2つです。

- 対象が振り返っている間に視線を遮れない
- 対象との距離管理に失敗し、警戒ゲージが最大になる

## 起動方法

依存関係はありません。ブラウザだけで動きます。

macOS では次のコマンドで起動できます。

```bash
open index.html
```

または、`index.html` をブラウザに直接ドラッグして開いてください。

## 操作

| キー | 動作 |
| --- | --- |
| `W` / `↑` | 前進する。遮蔽物や街並みが手前へ流れます |
| `S` / `↓` | 後退する。遮蔽物やアイテムが奥へ戻ります |
| `A` / `←` | 左へ移動 |
| `D` / `→` | 右へ移動 |
| `Space` | 近くの遮蔽物に隠れる |

## 画面の見方

- `DIST`: 対象との近さを示します。近づくほどバーが伸びます
- `ALERT`: 警戒ゲージです。最大になると失敗です
- `ITEMS`: 回収した落とし物の数です
- `TAILING`: 通常の尾行状態です
- `WARNING`: 対象が振り返る直前です
- `WATCH`: 対象が振り返っている状態です
- `HIDDEN`: 遮蔽物に隠れている状態です

## プレイのコツ

- 対象の近くに寄りすぎないようにしてください
- 対象が落としたアイテムは、前進して近づき、横位置を合わせると回収できます
- 電信柱やポストなどの遮蔽物は通り抜けられません。左右に避けてください
- `WARNING` が出たら、近くの遮蔽物へ移動してください
- 遮蔽物の足元に黄色い輪が出たら、`Space` で隠れられます
- 対象と自分の間に遮蔽物がある場合も、視線を遮れます
- 対象がこちらを見ている間は、隠れたまま動かないのが安全です

## 実装済み

- 奥行きのある疑似3Dの二次元 Canvas 画面
- 中央照準、周辺視界マスク、手元シルエットによる一人称視点UI
- 視界に大きく入る遮蔽物と対象表示
- 距離に応じて大きさと位置が変わる尾行対象
- 尾行対象が落とすアイテムと回収クリア条件
- 振り返りイベントと予兆表示
- 電信柱、ポスト、看板、自販機の遮蔽物
- 遮蔽物付近での隠れ判定と、遮蔽物による視線遮断判定
- 停止、左右移動、急ぎ足を含む対象の行動パターン
- 距離メーター、警戒ゲージ、発見/成功判定
